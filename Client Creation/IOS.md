# IOS
Simple Yet Effective tutorial by Wings 
To create IOS Client for your GDPS
- Download this [IPA file](https://gdwings.141412.xyz/library/apps/geometrydash.php) 

- Rename file extension to ".zip"

- Extract said file

- Navigate to file Payload /GeometryJump(folder) / GeometryJump (file)

- Swap servers with a Hex Editor 

- Compress it into a zip file

-  Rename the file extension to ".IPA"

Done 

Enjoy Its that simple since Noxicloud doesn't officially support IOS versions of your gdps we cannot help you.
